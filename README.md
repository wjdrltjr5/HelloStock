# HelloStock
***
주식 정보 제공과 커뮤니티 기능을 제공하는 웹입니다.

## 주요 기능
***
- 로그인
- 회원가입
- 게시판 
- 댓글
- 주식정보
- 관심주식
- 마이페이지
